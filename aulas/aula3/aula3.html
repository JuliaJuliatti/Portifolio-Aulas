Sobre a Aula

Esta aula cobriu assuntos de DOM (Document Object Model), que é fundamental para manipulação de elementos HTML com JavaScript. Esse conteúdo foi passado pelas meninas Ana e Isa.

Detalhes Aprendidos

  • Como selecionar elementos do DOM usando getElementById e querySelector.
  • Modificação de conteúdo e estilo de elementos HTML.
  • Adição e remoção de elementos do DOM.
  • Eventos e manipulação de eventos com JavaScript.
